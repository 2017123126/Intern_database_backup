**LoRA - faster way to fine-tune Stable Diffusion**

LoRA stands for Low-Rank Adaptation, a mathematical technique to reduce the number of parameters that are trained

LoRA has a few differences from DreamBooth that make it especially appealing as an alternative:

- **Faster training**: Training a new concept with LoRA takes just a few minutes.
- **Smaller outputs**: Trained LoRA outputs are much smaller than DreamBooth outputs. This makes them easier to share, store, and re-use.
- **Multiple concepts**: You can combine multiple trained concepts in a single image. (This feature is still experimental, but we're working on improving it. ğŸ§ª)
- **Faster image generation**: When you train your own DreamBooth model on Replicate, the model only stays warm when you're actively using it. With LoRA, you're not running your own model, but rather running the oneÂ [cloneofsimo/lora](https://replicate.com/cloneofsimo/lora)Â model, which is always on and ready to serve predictions.
